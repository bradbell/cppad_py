Solved¶

swig¶

If the error message below occurs, try installing http://www.swig.org/ on you system:

        FileNotFoundError: [Errno 2] No such file or directory: 'swig'

The following message as also been seen:

        Error: Unable to find 'python.swg'

In this case try installing swig-python .

Permissions¶

If you try to do a system wide install, and do not have permission, you will get the following message:

        ERROR: Could not install packages due to … Permission denied:

Try install in your local user space. For example, if you are using pip,

        pip install -i https://test.pypi.org/simple/ cppad_py --user --verbose

CppAD Library Missing¶

If the error message below occurs, it is because the CppAD library is not in your LD_LIBRARY_PATH :

        ImportError: libcppad_lib.so … can not open shared object file

If you run the setup.py script directory, or if you included the --verbose option in your pip command, there should be text at the end that tells you how to modify your LD_LIBRARY_PATH . If you have a Mac, you may instead need to modify DYLD_LIBRARY_PATH .

Fortify Source¶

If you set build_type to debug , you may get the following warning during the build:

        #warning _FORTIFY_SOURCE requires compiling with optimization

This is a problem with the python setuptools, one can un-define a macro, but it does not remove a original definition.

Travis¶

The following message occurs on a Travis system of unknown configuration; i.e., it has not been reproduced outside of Travis.

        return alpha * p * a_exp( - z * z ) / numpy.sqrt(numpy.pi)[0m
                TypeError: unsupported operand type(s) for /: 'a_double' and 'float'

Note the same code works on other systems. The .travis.yml file is included below:

        language: python
        python:
                - "3.7"
        install:
                - sudo apt-get install swig
                - python setup.py install
                - pip install -i https://test.pypi.org/simple/ cppad_py
        script:
                - pytest tests
                - make examples
