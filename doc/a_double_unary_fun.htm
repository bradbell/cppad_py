Unary Functions with AD Result

Syntax
ax
fun
ay
Example

Syntax
ay = ax.fun()
ax
This object has prototype
    const a_double& ax
 This is the argument for the function evaluation.

fun
This specifies which function is being evaluated and is one of following value: abs, acos, asin, atan, cos, cosh, erf, exp, fabs, log, sin, sinh, sqrt, tan, tanh. 2DO: Add the C++11 functions asinh, acosh, atanh, expm1, and log1p to this list.

ay
The result object has prototype
    a_double ay
 and is the specified function evaluated at the specified argument; i.e.,
    ay = fun( ax )
