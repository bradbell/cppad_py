Syntax

Purpose
The routine can be used with ad_double

f
This is a function that evaluates the ordinary differential equation using the syntax
     yp = f( t , y )
 where t is the current time, y is the current value of @(@ y(t) @)@, and yp is the current derivative @(@ y^{(1)} (t) @)@.

t_all
This is a numpy vector of time values at which the solution is calculated. The type of its elements can be float or ad_double. It must be either monotone increasing or decreasing. A single Runge-Kutta step is used to calculate the value at the next time given the value at the current time.

y_init
This is the value of @(@ y(t) @)@ at the initial time t_all[0] as a numpy vector. The type of its elements can be float or ad_double.

y_all
This is the approximate solution for @(@ y(t) @)@ at all of the times specified by t_all as a numpy array. The value y_all[i, j] is the value of the j-th component of @(@ y(t) @)@ at time t_all[i] . This solution is 4-th order accurate in time @(@ t @)@; e.g., if @(@ y(t) @)@ is a polynomial in @(@ t @)@ of order four or lower, the solution has no truncation error, only round off error.

from runge4_step import runge4_step
def multi_step(f, t_all, y_init ) :
     import numpy
     import copy
     dtype      = type(y_init[0])
     n_var      = y_init.size
     n_step     = t_all.size - 1
     y_all      = numpy.empty( (n_step+1, n_var), dtype = dtype  )
     y1         = y_init
     t1         = t_all[0]
     y_all[0,:] = y1
     for i in range(n_step) :
          y0            = y1
          t0            = t1
          t1            = t_all[i+1]
          t_step        = t1 - t0
          y1            = runge4_step(f, t0, y0, t_step)
          y_all[i+1,:]  = copy.copy(y1)
     return y_all
