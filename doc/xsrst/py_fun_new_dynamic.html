py_fun_new_dynamic¶

\(\newcommand{\B}[1]{ {\bf #1} }\) \(\newcommand{\R}[1]{ {\rm #1} }\)

New Dynamic Parameters¶

  • Syntax

  • f

  • dynamic

    • size_order

  • Example

Syntax¶

f . new_dynamic ( dynamic )

f¶

This is either a d_fun or a_fun. The independent dynamic parameters are changed to have the specified values. The other dynamic parameters are then computed.

dynamic¶

If f is a d_fun ( a_fun ) object, dynamic is a numpy vector with float ( a_double ) elements and its size must be the same as the size of dynamic in the corresponding call to independent . It specifies new values for the dynamic parameters in f .

size_order¶

After this call, f_size_order() is zero.

Example¶

See fun_dynamic_xam_py
