a_double Assignment Operators
#############################

.. meta::
   :keywords: syntax

.. index:: syntax

.. _a_double_assign@Syntax:

Syntax
******
| *ax* *op* *ay*
| *aw* *op* *y*

.. meta::
   :keywords: op

.. index:: op

.. _a_double_assign@op:

op
**
The assignment operator *op* is one of the following:

.. csv-table::
   :widths: 20, 20, 20

   *op*   , *ax* Syntax       , *aw* Syntax
   ``=``  , *ax* = *ay*       , *aw* = *y*
   ``+=`` , *ax* = *ax* + *ay*, *aw* = *aw* + *y*
   ``-=`` , *ax* = *ax* - *ay*, *aw* = *aw* - *y*
   ``*=`` , *ax* = *ax* * *ay*, *aw* = *aw* * *y*
   ``/=`` , *ax* = *ax* / *ay*, *aw* = *aw* / *y*
